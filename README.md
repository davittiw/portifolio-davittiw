### [PT-BR]
## Descrição
Esse portifólio foi criado para que minha trajetória como desenvolvedor full-stack fosse compartilhada e mais vista por interessados. Utilizei Tailwind e React para o desenvolvimento dele para também aprimorar minhas habilidades no desenvolvimento front-end com techs modernas.

### Tecnologias Utilizadas
- [TailwindCSS](https://tailwindcss.com/docs/installation/using-vite)
- [React](https://react.dev/learn)

## Como rodar o projeto
- Clone o repositório: ```git clone https://github.com/davittiw/portifolio-davittiw.git```
- No terminal, rode ```npm install``` ou ```npm i```
- rode ```npm run dev```
- Clonado!

## Funcionalidades
- Internacionalização [PT/EN]
- Animações e transições suaves
- Fórmulário de contato funcional
- Avaliação com estrelas interativas
- Design responsivo com Tailwind

### Estrutura de Pastas
- `./locales`: tem os arquivos json para a internacionalização
- `./src/components`: componentes da página
- `./src/assets`: pasta fonts onde há a fonte utilizada no portifólio
- `./public`: local das fotos utilizadas

### [EN-US]
## Description  
This portfolio was developed to share my journey as a full-stack developer and make it more accessible to interested professionals and recruiters. I used **React** and **TailwindCSS** to improve my front-end development skills using modern technologies.

## Technologies Used  
- [TailwindCSS](https://tailwindcss.com/docs/installation/using-vite)  
- [React](https://react.dev/learn)

## How to Run the Project  
- Clone the repository:  ```git clone https://github.com/davittiw/portifolio-davittiw.git```
- Install dependencies: ```npm install``` ou ```npm i```
- Run the development server: ```npm run dev```
- Done!

## Features
- Internationalization (PT/EN)
- Smooth animations and transitions
- Functional contact form
- Interactive star rating system
- Responsive design with TailwindCSS

### Folder Structure
- `./locales`: contains .json files for internationalization
- `./src/components`: reusable page components
- `./src/assets`: contains the fonts folder with the custom font used in the portfolio
